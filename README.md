
# Tab Sections using HTML, CSS & JavaScript

This project demonstrates a **Tab Sections UI component** where clicking on a tab button dynamically displays the related content while hiding the others. It is built using **pure HTML, CSS, and JavaScript** without any external libraries.

## Features

* Interactive tab navigation
* Active tab highlighting
* Smooth content switching
* Fully responsive design
* Clean and beginner-friendly code
* No external libraries used

## How It Works

* Each button represents a tab.
* When a button is clicked:

  * The related content section becomes visible.
  * Other content sections are hidden.
  * The clicked button changes its background color to show it is active.
  * Previously active buttons return to normal color.
